# Digital Twin Automation Simulation (Unity)

## 프로젝트 개요

자동화 생산 라인의 흐름을  
**디지털 트윈 형태의 시뮬레이션으로 표현한 Unity 기반 개인 프로젝트**입니다.

공정 내 아이템 이동, 센서 감지, 로봇 작업, 상태 전환 과정을  
단순 시각화가 아닌 **상태(State)와 이벤트(Event) 구조**로 모델링했습니다.

---

## 기획 의도

- 로그·데이터로만 존재하던 공정 상태를 **시각적으로 표현**
- 공정이 멈추거나 밀리는 지점을 **구조적으로 이해 가능하게 설계**
- 자동화 시스템의 흐름을 **코드 구조 자체로 설명**하는 시뮬레이션 구현

---

## 설계 방향

- 연출 중심이 아닌 **시스템 동작 중심 설계**
- 상태 기반(State-driven) 공정 흐름
- 이벤트(Event) 중심 제어 구조
- 비동기 처리(Coroutine / Queue)를 통한 현실적인 공정 표현

---

## 핵심 구성

- Zone 기반 공정 구조 (Zone 1 → Zone 2 → Zone 3)
- 센서 감지 → 이벤트 전달 → 로봇 동작 흐름
- Pick & Place 로봇 제어 시퀀스
- Queue 기반 아이템 대기 처리
- 공정 상태(Running / Stop 등) 관리

---

## 구현 범위 (현재)

- 기본 공정 흐름 및 로봇 동작 구현
- 상태·이벤트 구조 검증 완료
- 구조 설계 중심 프로토타입 단계

※ 빌드 및 UI 확장은 추후 진행 예정

---

## 활용 포인트

- Unity를 **게임 엔진이 아닌 시뮬레이션 도구로 활용**
- 복잡한 공정 로직을 **상태·이벤트 구조로 정리**
- 공정 병목 및 흐름을 **직관적으로 확인 가능한 구조**

---

## 실행 방법

1. GitHub 저장소를 Clone 또는 Download
2. Unity Hub 실행
3. `Add` → 프로젝트 폴더 선택
4. `SampleScene` 열기
5. Play 모드에서 시뮬레이션 확인

---

### 요약

- 자동화 공정 흐름을 시각화한 디지털 트윈 시뮬레이션
- 상태·이벤트 중심 구조 설계
- 시스템 동작 이해를 위한 Unity 기반 모델링
