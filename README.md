# Digital Twin Automation Simulation (Unity)

## 프로젝트 개요

자동화 생산 라인의 공정 흐름을  
**디지털 트윈 형태로 시뮬레이션한 Unity 기반 개인 프로젝트**입니다.

아이템 이동, 센서 감지, 로봇 동작, 공정 정지·재개 과정을  
단순 연출이 아닌 **상태(State)와 이벤트(Event) 구조**로 구현했습니다.

---

## 기획 의도

- 로그·데이터로만 존재하던 공정 흐름을 시각적으로 이해
- 공정 정지·병목 상황을 구조적으로 설명 가능한 형태로 모델링
- 자동화 시스템의 동작을 코드 구조 자체로 표현

---

## 핵심 설계

- 상태(State) 기반 공정 제어 (Plant / Zone 단위)
- 이벤트(Event) 중심 공정 연결 구조
- Queue 기반 아이템 대기 및 처리
- Coroutine을 활용한 비동기 공정 흐름

---

## 구현 요소

- Zone 1 → Zone 2 → Zone 3 공정 흐름
- 센서 → Sink → Buffer → Pick & Place 로봇 시퀀스
- 공정 Stop / Resume 시 Queue 유지 및 재처리
- PickUpSlot 점유 관리 및 로봇 이벤트 트리거
- Observer Camera 모드 (공정 관찰용 자유 시점)
- 공정 이벤트 및 세션 데이터를 JSON 구조로 기록 가능

---

## Fault 및 테스트 구조

- Fault는 테스트용 외부 이벤트로 설계
- 특정 Zone만 정지하여 공정 흐름 검증
- 전체 시스템 종료 없이 상태 전환 및 복구 가능

---

## 프로젝트 성격

- 실제 공장을 1:1 재현하는 것이 목적은 아님
- 자동화 공정의 **상태 흐름과 제어 구조 검증**이 핵심
- 시각적 완성도보다 시스템 설계와 로직 구조에 집중

---

## 실행 방법

1. 저장소 Clone 또는 Download
2. Unity Hub → 프로젝트 추가
3. `SampleScene` 실행
4. Play 모드에서 시뮬레이션 확인
